¿Qué es un Protocolos Http?

El http (inglés HyperText Transfer Protocol o Protocolo de Transferencia de Hiper Textos) es el protocolo de transmisión de información de la World Wide Web, es decir, el código que se establece para que el computador solicitante y el que contiene la información solicitada puedan “hablar” un mismo idioma a la hora de transmitir información por la red.

¿Qué es un Query String?

En español cadena de consulta,este término generalmente se utiliza para hacer referencias a una interacción con una base de datos. Es la parte de la URL que contiene los datos que deben pasar a las aplicaciones web.

¿Qué es un Verbos HTTP?

Los verbos http son indicadores de acciones que queremos que realize un servidor y esots son: GET, POST, PUT, PATCH, DELETE, HEAD, CONNECT, OPTIONS y TRACE. Cada uno indica una acción diferente a la que el servidor debe responder.

¿Qué es un DNS?

Es una base de datos distribuida cuya función es traducir los nombres de dominio en las redes,con esto se quiere decir que actuan como un traductor de direcciones ip para crear nombres que se puedan recordar como los dominios.

¿Qué es un CDN?

Una red de distribución de contenido (CDN) es un grupo de servidores repartidos en distintas zonas geográficas que aceleran la entrega del contenido web al acercarlo a los usuarios.on el fin de maximizar el ancho de banda para el acceso a los datos de clientes por la red.
